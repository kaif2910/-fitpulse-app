TECHNICAL PROJECT REPORT

FitPulse: Daily Workout Tracker

Angular 18 | Fitness Management System

1. Project Aim

The FitPulse Tracker is designed to help users log their daily gym activities. It uses Angular's reactive nature to track sets, reps, and workout duration. The project demonstrates how technical concepts can be applied to health and fitness monitoring.

2. Technical Features

  • Two-Way Binding: Used for entering workout names and reps.
  • Custom Directive: IntensityHighlight adds a dynamic glow based on workout stress level.
  • Custom Pipe: WorkoutTime formats training duration from seconds to minutes.

3. Source Code Implementation

3.1 Workout Logic (workout.ts)

export class WorkoutComponent { workoutName: string = ''; durationSeconds: number = 3600; // 1 hour workouts = [ { name: 'Pushups', reps: 50, intensity: 'High' }, { name: 'Yoga', reps: 1, intensity: 'Low' } ]; addWorkout() { this.workouts.push({ name: this.workoutName, reps: 0, intensity: 'Medium' }); } }

3.2 Custom Timer Pipe (workout-time.pipe.ts)

@Pipe({ name: 'workoutTime', standalone: true }) export class WorkoutTimePipe implements PipeTransform { transform(seconds: number): string { const mins = Math.floor(seconds / 60); const secs = seconds % 60; return `${mins}:${secs < 10 ? '0' : ''}${secs}`; } }

3.3 Template View (workout.html)

<div class="fitness-dashboard"> <h1>Total Time: {{ durationSeconds | workoutTime }}</h1> <input [(ngModel)]="workoutName" placeholder="New Exercise"> <div class="exercise-card" *ngFor="let w of workouts" appIntensity> <h3>{{ w.name | uppercase }}</h3> <p>Target: {{ w.reps }} reps</p> </div> </div>
